## 技术栈
- SpringBoot
- Mybatis-plus
- MySQL
- Satoken
- Vue

## 要点
- 用户登录、注册
- 图书浏览，查询，加购
- 购物车管理，结算，会员打折，下单
- 个人订单记录，详情
- 书籍管理
- 书籍分类管理
- 订单管理
- 用户管理，会员授予
- 数据统计
- 每日折扣设置

::: warning
代码高度规范、精简，可继续扩展~
:::
