# 📚超市信息管理系统

## 技术栈
- 开发语言：Java
- 数据库：MySQL
- 依赖：JDBC
- 交互：Swing

## 实现功能
- 管理员登录
- 进货
- 售货
- 信息查询（库存、进货、售货）
